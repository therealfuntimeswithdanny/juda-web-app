How to Add Notes via URL

You can add a new note to Juda Notes directly via a URL query parameter. Simply append the parameter ?add=Your%20Note%20Text to the URL of your main app.

For example, if your main notes page is hosted at https://judaapp.madebydanny.uk, navigating to:

https://judaapp.madebydanny.uk?add=Hello%20World

will automatically create a new note with the content "Hello World".
